# Focus Mute

一个简洁的 Chrome 浏览器扩展，自动静音所有非活动标签页，让你专注于当前内容。

## 功能特性

- **一键切换**：通过简单的开关按钮启用/禁用专注模式
- **智能静音**：仅保留当前活动标签页的声音，静音其他所有标签页
- **状态清晰**：图标颜色明确显示当前状态（彩色=启用，灰色=禁用）
- **Manifest V3**：符合最新的 Chrome 扩展标准
- **轻量高效**：最小化性能影响

## 安装方法

### 方式一：开发者模式安装（推荐用于测试）

1. 下载或克隆本项目到本地
2. 打开 Chrome 浏览器，访问 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目文件夹（包含 manifest.json 的目录）
6. 扩展安装完成！

### 方式二：打包安装

1. 在 `chrome://extensions/` 页面点击"打包扩展程序"
2. 选择项目文件夹
3. 生成 .crx 文件后拖拽到浏览器安装

## 使用说明

1. **启用专注模式**
   - 点击浏览器工具栏中的 Focus Mute 图标
   - 在弹出窗口中点击"启用专注模式"按钮
   - 图标变为彩色，表示已启用

2. **切换标签页**
   - 切换到任何标签页时，该标签页自动取消静音
   - 其他标签页自动静音

3. **禁用专注模式**
   - 点击扩展图标，然后点击"禁用专注模式"
   - 所有标签页恢复正常声音状态
   - 图标变为灰色

## 自定义图标

项目包含了一个简单的图标生成器。如果你想要更美观的图标：

1. 在浏览器中打开 `icons/icon-generator.html`
2. 点击"下载所有图标"按钮
3. 将下载的图标文件替换 `icons` 目录中的对应文件

## 技术细节

- **Manifest V3**：使用最新的 Service Worker 架构
- **Chrome Tabs API**：用于访问和操作标签页
- **Chrome Storage API**：持久化存储扩展状态
- **Chrome Action API**：管理扩展图标和弹出窗口

## 权限说明

- `tabs`：访问和操作浏览器标签页的静音状态
- `storage`：保存扩展的启用/禁用状态
